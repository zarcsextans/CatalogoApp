# 🎮 CatalogoApp

CatalogoApp es una aplicación web desarrollada en ASP.NET Core MVC que permite gestionar un catálogo de videojuegos, incluyendo su visualización, registro, filtrado por género y sistema de reseñas de usuarios.

---

## 🚀 Funcionalidades

- 📋 Visualización de catálogo de videojuegos
- 🔍 Filtro por género
- ➕ Agregar nuevos videojuegos
- 🔐 Registro e inicio de sesión de usuarios
- ⭐ Sistema de reseñas con calificaciones
- 💾 Persistencia de datos usando archivos JSON

---

## 🛠️ Tecnologías utilizadas

- ASP.NET Core MVC
- C#
- Razor Views
- HTML, CSS, Bootstrap
- JSON como almacenamiento local
- Session para autenticación básica

---

## 📂 Estructura general

- **Models**: Item, Usuario, Review
- **Controllers**: CatalogoController, LoginController, HomeController
- **Views**: Catálogo, detalle, login, registro
- **Helpers**: manejo de lectura y escritura JSON

---

## ⚠️ Nota de seguridad

Este proyecto es académico y utiliza almacenamiento local en JSON, por lo que no es seguro para producción real. Las contraseñas se guardan en texto plano únicamente con fines educativos.
